# Hack&Slash RPG 소개
탑 뷰 형식의 스타일리쉬한 Hack & Slash 장르의 싱글플레이 RPG 게임입니다. 던전을 탐험하여 몬스터를 사냥하거나 숨겨진 상자를 찾아서 아이템을 파밍하고 장비를 장착해 캐릭터 스탯을 올려서 최종적으로 보스 몬스터를 처치하여 승리하는 레벨로 구성되어있습니다.
